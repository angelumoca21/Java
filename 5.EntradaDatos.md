# 5.Entrada de datos
Scanner es una clase de Java que permite leer datos que el usuario escribe en la consola.

1.Importar:
```
import java.util.Scanner;
```

2.Crear objeto:
```
Scanner sc = new Scanner(System.in);
```

3.Especificar el tipo de dato de interés:
- String
    ```
    System.out.print("Escribe tu nombre completo: ");
    String nombreCompleto = sc.nextLine();
    ```
- Palabra
    ```
    System.out.print("Escribe tu apellido: ");
    String apellido = sc.next();
    ```

- int
    ```
    System.out.print("Escribe tu edad: ");
    int edad = sc.nextInt();
    ```

- double
    ```
    System.out.print("Escribe tu estatura: ");
    double estatura = sc.nextDouble();
    ```

- char
    ```
    System.out.print("Escribe una letra: ");
    char letra = sc.next().charAt(0);
    ```

## 5.1.Problema común con nextLine()
Si se usa `nextInt()` antes de `nextLine()`, puede saltarse la lectura.

Solución:
```
int edad = sc.nextInt();
sc.nextLine(); // limpiar buffer
String nombre = sc.nextLine();
```
