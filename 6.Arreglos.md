# 6.Arreglos

Un arreglo es una estructura que guarda varios valores del mismo tipo dentro de una sola variable.
![Arrays en Java](https://images.shiksha.com/mediadata/ugcDocuments/images/wordpressImages/2022_05_Arrays-in-Java-1.jpg)

## 6.1.Declaración de arreglo
```
tipo[] nombre = new tipo[n];

n = número de elementos
```

## 6.2.Inicialización y declaración de arreglo
```
tipo[] nombre = {valor1, valor2, ... , valorn};
```

## 6.3.Longitud de un arreglo
Para obtener la longitud de un arreglo, podemos acceder al campo `length`.

## 6.4.Recorrer arreglo 
Para imprimir o llenar un arreglo se suelen usar ciclos.
