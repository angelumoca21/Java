# Lenguaje de Programación Java

### *Duración: 60 horas (30 sesiones [Lunes a jueves] y 15 sabatinas)*

## Objetivo general

El estudiantado empleará los conocimientos en lenguaje de programación JAVA en los procesos de reunión, análisis e interpretación de datos a través de prácticas y ejercicios de programación.

## Evaluación

- 50% Actividades en sesiones
- 50% Proyecto final
