## 2.1.Tipos de datos simples o primitivos
- Entero (int, byte, short, long)
- Decimales o de punto flotante (float, double)
- Booleanos (boolean)
- Carácter (char)

## 2.2.Tipos de datos no primitivos 
- Cadenas de texto (String)
- Arreglos (Arrays)
- Clases

## 2.3.Variables
Una variable es un espacio en memoria que se utiliza para almacenar un valor que puede cambiar, cada variable tiene un nombre o identificador.

### 2.3.1 Definición de variables
```
tipo identificador = valor;
``` 

Por ejemplo:
```
int edad = 28; //Entero
float precio = 9.99; //Flotante
char opcionEscogida = 'A'; //Carácter
boolean inscrito = true; //Booleano
```

## 2.4.Reglas para nombrar variables
✔ Permitido:
- Letras
- Números (no al inicio)
- Guion bajo `_`

❌ No permitido:
- Espacios
- Símbolos
- Palabras reservadas (int, class, public)

## 2.5.Imprimir variables
```
String nombre = "Angel";
int edad = 25;
double promedio = 8.7;

System.out.println("Nombre: " + nombre);
System.out.println("Edad: " + edad);
System.out.println("Promedio: " + promedio);
```
