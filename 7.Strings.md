# 7.Strings

Un String es un texto.

En Java:
- String NO es un tipo primitivo es una clase. Por eso tiene métodos.

## 7.1.Crear String
```
String nombre = "texto";
```

## 7.2.Métodos de String

### 7.2.1.Longitud
Para obtener la longitud del un String.
```
nombreVariable.length();
```

### 7.2.2.Convertir mayúsculas y minúsculas
```
nombreVariable.toUpperCase();
nombreVariable.toLowerCase();
```

### 7.2.3.Buscar texto
```
nombreVariable.contains("cadenaABuscar");
```

### 7.2.4.Inicia con
```
nombreVariable.startsWith("a");
```

### 7.2.5.Comparar Strings
```
nombreVariable.equals(nombreOtraVariable);
```

### 7.2.6.Recortar String
```
nombreVariable.substring(0,2);
```

### 7.2.7.Obtener un caracter
```
nombreVariable.charAt(#indice);
```

### 7.2.8.Reemplazar texto
```
nombreVariable.replace("textoAReemplazar","TextoQueReemplazara");
```

### 7.2.9.Eliminar espacios
```
nombreVariable.trim();
``` 

### 7.2.10.Posición de elemento
```
nombreVariable.indexOf("elemento")
```
