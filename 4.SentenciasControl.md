# 4.Sentencias de control

El compilador de Java ejecuta el código de arriba a abajo. Las declaraciones en el código se ejecutan según el orden en que aparecen. Sin embargo, Java proporciona sentencias que se pueden usar para controlar el flujo del código Java.

## 4.1.Sentencias de decisión (Condicionales)
Permiten que el programa tome decisiones.

### 4.1.1.Sentencia **if**
La forma más simple de un estamento condicional es un if (del inglés si)
```
if (condicion) {
    // código que se ejecuta si la condición es true
}
```

### 4.1.2.Sentencia **if - else**
```
if (condicion) {
    // si se cumple
} else {
    // si NO se cumple
}
```

### 4.1.3.Sentencia **if - else if - else**
```
if (condicion1) {
    // caso 1
} else if (condicion2) {
    // caso 2
} else {
    // caso final
}
```

### 4.1.4.Sentencia **switch**
```
switch (variable) {
    case valor1:
        // código
        break;
    case valor2:
        // código
        break;
    default:
        // código
}
```

## 4.2.Sentencias de repetición (ciclos)
Permiten que el programa repita instrucciones varias veces.

### 4.2.1.While
Se usa cuando no sabemos exactamente cuántas veces se repetirá algo.
```
while (condicion) {
    // código que se repite
}
```
### 4.2.2.Do-While
Similar al while, pero se ejecuta al menos una vez.
```
do {
    // código
} while (condicion);
```

### 4.2.3.For
El más usado cuando sabemos cuántas veces repetir.
```
for (inicio; condicion; avance) {
    // código
}
```
